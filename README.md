# Cecelia Personal Website

Personal website for Cecelia, built with React and Vite, and deployed with GitHub Pages.

## Tech Stack

- React
- Vite
- GitHub Pages
- GitHub Actions

## Features

- Personal profile and education
- Featured projects and full project archive
- Project detail pages with Markdown content
- Search and filter for all projects
- Responsive layout
- Light and dark theme toggle

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deployment

This site is deployed through GitHub Actions.

Push changes to the `main` branch to trigger deployment.
